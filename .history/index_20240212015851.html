Projects

Here are some of the most relevent projects I have made until now. I wanted to start with the project I am most proud of. I programmed a syntax highlighter that works the same way as the highlighters in IDEs like vs code, Atom, Sublime, etc. To do this, the program first tokenizes the inputted python file, then makes use of regular expressions to categorize each type of token and depending on the type of token, creates a copy of it in an html file in which it is given highlight color with CSS and thus each type of token ends up with the specific color of its category. Then you can open the html file in a browser and see the same python input file with the text highlighted in colors.

    What i learned

  • Reading and writing different types of files

  • Regular Expressions

  • Tokenization
